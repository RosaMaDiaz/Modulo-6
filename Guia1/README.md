###  📖 **Contenidos Abordados**
### **1. Formas de Aplicar CSS**
La guía explora las tres maneras de integrar estilos:  
  - Inline (En línea): Aplicado directamente en la etiqueta HTML mediante el atributo style.  
  - Internal (Interno): Definido dentro de la etiqueta <style> en el <head>.  
  - External (Externo): Archivo independiente .css vinculado mediante <link>, siendo esta la mejor práctica profesional.

### **2. Selectores Básicos**
Se practican los selectores fundamentales para controlar la apariencia: 
 - **Universal (*):** Aplica a todos los elementos (ideal para "resets"). 
 - **Etiqueta**: Selecciona todos los elementos de un tipo (ej. p, h1).
 - **Clase (.clase):** Permite reutilizar estilos en múltiples elementos.
 - **ID (#id):** Para elementos únicos y específicos dentro de la página

### **3. Especificidad y Prioridad**
Se analiza cómo el navegador resuelve conflictos cuando hay múltiples reglas:  
 - Estilo en línea (Máxima prioridad).  
 - ID.  
 - Clase.  
 - Etiqueta. 
 - Selector universal (Menor prioridad).
