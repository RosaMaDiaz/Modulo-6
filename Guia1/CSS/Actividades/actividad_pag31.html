Selector de clase (.clase)

El selector de clase en CSS permite aplicar estilos a uno o varios elementos HTML que comparten una misma clase. A diferencia del selector de etiqueta, que afecta a todos los elementos de un tipo, el selector de clase ofrece un mayor nivel de control, ya que permite seleccionar únicamente los elementos que tengan un atributo class específico. Esto facilita la reutilización de estilos y la personalización de diferentes partes de una página web sin afectar a todos los elementos.
