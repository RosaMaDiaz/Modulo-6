# Módulo 6: Diseño de Estructuras de Página Web (CSS)

Este repositorio contiene el compendio de prácticas y proyectos desarrollados a lo largo de las tres Guías de Laboratorio del Módulo 6, bajo la guía del docente **Lic. Ramón Morillo Ullola**. El objetivo principal es dominar el uso de CSS3 para transformar maquetas estructurales en interfaces web estéticas, organizadas, interactivas y adaptables (*responsive*).

---

## 📂 Estructura General del Módulo

### 📘 Guía 1: Introducción al CSS — Estilos, Fuentes y Formatos
* **Resultado de Aprendizaje (RA1):** Utiliza los elementos del CSS para configurar fuente, estilo y formato de una página web.
* **Conceptos Clave:**
    * Sintaxis básica de CSS (selectores, propiedades y valores).
    * Métodos de incorporación: Estilos en línea (*inline*), hojas internas (`<style>`) y hojas externas (`<link>`).
    * Manejo de fuentes, tipografías y alineación de textos básicos.
* **Proyecto Central:** Construcción de la base de un **Portafolio Académico** en HTML estructurado para documentar el progreso del estudiante.

### 📗 Guía 2: Propiedades de Estilo para Diseñar Contenido Web
* **Resultado de Aprendizaje (RA2):** Aplica propiedades de estilo para diseñar contenido web logrando legibilidad y orden visual.
* **Conceptos Clave:**
    * Formatos de color modernos (Nombres, Hexadecimal, RGB/RGBA, HSL/HSLA).
    * **El Modelo de Caja (*Box Model*):** Control exhaustivo de `margin`, `padding`, `border`, `content` y la propiedad fundamental `box-sizing: border-box`.
    * Propiedades de fondo (`background-color`, imágenes y degradados).
* **Proyecto Central:** Evolución y estilización del contenido del portafolio mediante un diseño de caja limpio, bordes definidos y espaciados proporcionales.

### 📙 Guía 3: Propiedades y Técnicas de Diseño en CSS (Layout Avanzado)
* **Resultado de Aprendizaje (RA3):** Utiliza propiedades y técnicas de diseño en CSS para organizar contenidos en columnas, estructurar interfaces de usuario y desarrollar páginas web adaptables (*responsive*).
* **Conceptos Clave:**
    * **Flexbox:** Maquetación unidimensional (ejes, alineación, distribución y control de espaciados con `gap`).
    * **CSS Grid:** Maquetación bidimensional estructurada utilizando áreas complejas (`grid-template-areas` y cuadrículas flexibles).
    * **Diseño Adaptable (*Responsive Design*):** Implementación de la etiqueta Viewport y uso de *Media Queries* (`@media`) para la visualización multi-dispositivo.
    * **Interactividad:** Aplicación estética de sombras (`box-shadow`), bordes redondeados (`border-radius`) y transiciones suaves con `:hover` y `transition`.
* **Proyectos Centrales:** Página de cuadrícula flexible, maquetación adaptable de un sitio educativo institucional y un catálogo de tarjetas interactivas de productos.

---

## 🛠️ Tecnologías y Metodologías Aplicadas
* **Marcado Estructural:** HTML5 Semántico.
* **Estilos y Layout:** CSS3 Avanzado (Flexbox, Grid Layout).
* **Diseño Responsivo:** Mobile-First, Media Queries y Breakpoints Dinámicos.
* **Buenas Prácticas:** Modularización de archivos CSS (`base.css`, `color.css`, etc.) y optimización de herencia en cascada.
