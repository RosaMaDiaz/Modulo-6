Actividad de Grid Moderno

CSS Grid Layout es un sistema de diseño bidimensional en CSS que permite estructurar páginas web en filas y columnas simultáneamente. Se activa aplicando display: grid a un contenedor, lo que facilita crear diseños complejos, ordenados y responsivos sin depender de tablas o múltiples divs flotantes, siendo ideal para layouts completos.

Conceptos Clave y Propiedades

  • Grid Container (Padre): Elemento con display: grid o display: inline-grid.
  • Grid Items (Hijos): Elementos directos dentro del contenedor que se posicionan automáticamente en la cuadrícula.
  • Estructura: Se define con grid-template-columns (columnas) y grid-template-rows (filas).
  • Unidad fr (Fracción): Representa una fracción del espacio disponible, útil para diseños flexibles.
  • Gap: La propiedad gap (o grid-gap) establece el espacio entre filas y columnas.

Ejemplo de CSS:
.contenedor {
display: grid;
grid-template-columns: repeat(3, 1fr); /* Crea 3 columnas iguales */
grid-template-rows: auto;
gap: 20px; /* Espacio entre celdas */
}

Aplicación en el desarrollo web

CSS Grid permite crear:

  • Diseños de páginas completas
  • Galerías de imágenes
  • Paneles estructurados
  • Interfaces modernas

Buenas prácticas generales

  • Usar repeat() para simplificar código
  • Definir tamaños de columnas y filas con unidades fraccionarias
  • Utilizar gap para crear espacio entre elementos
  • Planificar el diseño antes de implementarlo

¿Qué integra este ejemplo?

  • display: grid → activa el layout
  • grid-template-columns → estructura en 2 columnas
  • repeat() → simplifica el código
  • gap → separación uniforme
  • box-shadow → profundidad visual
  • border-radius → diseño moderno
