## 📖 Descripción

Esta guía práctica tiene como objetivo enseñar el uso de propiedades CSS para mejorar la presentación visual de páginas web. A lo largo de las actividades, se trabajan conceptos fundamentales como:

- Colores y estilos de texto
- Tamaños de fuente
- Tipografías
- Alineación de contenido
- Fondos y degradados
- Espaciado con margin y padding
- Modelo de caja (Box Model)
- Organización visual de páginas web

El participante desarrollará ejemplos prácticos y un proyecto final integrador aplicando HTML y CSS.

---

# 🧠 Competencias Desarrolladas

Al finalizar esta guía, el estudiante será capaz de:

✅ Aplicar colores y estilos al texto  
✅ Modificar tamaños de fuente utilizando diferentes unidades  
✅ Usar tipografías adecuadas para distintos contextos  
✅ Alinear contenido correctamente  
✅ Aplicar fondos y degradados  
✅ Utilizar margin y padding para organizar espacios  
✅ Comprender y aplicar el Box Model  
✅ Diseñar páginas web visualmente organizadas
